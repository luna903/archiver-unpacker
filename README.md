# ScriptLangs_2026.Task-2P.Template

## 2026: Задание для студентов МИИГАиК и ИПМиКН - Вариант 2 (простой, только Python)

Используя исключительно стандартную библиотеку напишите на Python 3.14 консольную утилиту архиватор/распаковщик. В качестве алгоритма должно быть доступно на выбор zstd  и bz2. Если источником архивации является директория - необходимо предварительно использовать tarfile. Выбор режима архивации и определяется автоматически расширением файла целевого архива (file.zst или file.bz2). Для параметров командной строки необходимо добавить справку средствами argparse. 
Требования к решению:
1. Разработанную утилиту разместите в Smartclass используя приглашение (появится позже), ссылку на ваш репозиторий приложите как текстовый ответ. НЕ присылайте код программы напрямую в moodle.
2. Обязательно добавьте Readme с указанием ключей и примеров параметров запуска утилиты.
3. Приложите скриншоты с демонстрацией архивации и распаковки файлов и директорий с выводом времени работы (4 шт соответственно)
4. решение должно быть сделано самостоятельно и студент должен быть готов пояснить любой фрагмент кода
